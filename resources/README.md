# Capacitor Asset Placeholders

These files are placeholders for native app branding:

- `icon.png` -> app icon source (square, high resolution, e.g. 1024x1024)
- `splash.png` -> splash screen source (recommended wide canvas, e.g. 2732x2732)

Current placeholders were copied from existing web icons so the Android wrapper has valid defaults.

Before Play Store release, replace both files with final branded assets and regenerate native resources.
